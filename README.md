The IoT -based heart monitoring system using electrocardiogram (ECG) successfully
demonstrates a low cost, portable and efficient solution for real -time heart monitoring. Using
the AD8232 ECG sensor in combination with an Arduino microcontroller, the system is able
to accurately capture the electrical activity of the heart and display it as a waveform. This
allows users and health professionals to observe vital components such as P WAVE, QRS
complex and t wave, which are essential for detecting heart abnormalities such as irregular
arrhythmias or heartbeat. One of the main achievements of this project is the real -time view
of ECG data using Python, which allows users to monitor cardiac activity directly from a
computer through USB serial communication.
